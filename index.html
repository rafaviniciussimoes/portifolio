PROJETOS

API Eventos

A API Eventos tem como objetivo a manipulação dos seguintes recursos: usuários, eventos e compras. A api fornece um endpoint para cadastro de um usuário administrador do sistema, que por sua vez, será o único a ter acesso às rotas de manipulação dos recursos por meio de autenticação JWT. Possui endpoints para listagens, cadastros, atualizações e deleções dos recursos. A persistência dos dados é realizada em um banco de dados PostgreSQL, envolta em um container Docker.
